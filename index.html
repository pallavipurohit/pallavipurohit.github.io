What's the website about?

This website is a personal project of mine to showcase my knowledge and skills in web development. The webiste is divided into three different parts. The website opens up to the HOME page which initially gives a brief summary of the website using graphics and then in textual form followed by some of my favourite videos on youtube. The ABOUT-ME page is the profile and works as my resume in a very interactive form. BLOG sections is the place where you and I can share our ideas. The Blog page is designed using soft colors for easy reading.
